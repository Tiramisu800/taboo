# taboo 

Career-oriented website using nodejs + expressjs + mongoDB

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) 

```sh
$ git clone ...
$ cd taboo
$ npm install
$ npm start
```
Your app should now be running on [localhost:3000](http://localhost:3000/).
